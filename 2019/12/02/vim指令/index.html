    vim 使用技巧

    vim 是一个功能强大的屏幕文本编辑器，是Linux/unix 上最常用的文本编辑器，他的作用是建立、编辑、显示文本文件
    vim没有菜单，只有命令
    vim 的三个模式

    • 命令模式（键盘上的每个按键都是一个命令）；
    • 编辑模式（在命令模式下输入 ：即进入编辑模式）
    • 插入模式 （在命令模式下按 i ，即可进入插入模式）

    进入命令模式

    vi fileName

    编辑模式

    命令模式输入 ：即进入 编辑模式
